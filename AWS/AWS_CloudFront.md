# 💡 Amazon CloudFront

* 사용자들에게 좀 더 빠르게 컨텐츠를 제공하기 위한 서비스
  * 여기서 컨텐츠란 웹 페이지의 구성(html, css 등)과 이미지, 동영상 등을 의미
  * 전 세계의 **Edge Location** 에서 보다 빠르게 컨텐츠를 전달
* 글로벌 캐싱 서비스


## 📌 CloudFront 주요 기능

* CDN
* Caching
* Global Edge Location
* Static Website Hosting

## 📌 CloudFront 동작 원리

* Viewer & Origin
* Behavior
* Invalidation
